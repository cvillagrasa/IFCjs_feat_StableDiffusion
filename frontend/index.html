USAGE INSTRUCTIONS

Firstly, load an IFC file and use your mouse buttons to position the 3D view as preferred. While doing so, keep in mind that white spaces will be inpainted by the AI model. Then, fill the input fields below, which will guide the image generation process. When satisfied, click on the Generate button and see the result ;)
